In order to run the program in local environment. Please have below configuration settings.

1. Import the project as is using maven project
2. Run the mvn clean install and it will download all the dependent library 
3. Perform mvn eclipse:eclipse (optional) to set the eclipse class path as per the library
4. Directly right click on DeDup.java or DeDupArrayTest.java to see the result
